# CafeCalc — Инвестиционный Аналитик ИС Онлайн Кафе

Это мини-информационная система для оценки экономической эффективности внедрения ИС Онлайн Кафе (TCO, ROI, Payback Period, анализ чувствительности, сравнение локального и облачного сценариев).

## Структура репозитория

- `/docs` — методические материалы и навигация
- `/project` — учебные артефакты (документы, DevOps, скелеты API)
- `/templates` — шаблоны отчётов и документации
- `.github/workflows` — CI-конфигурация

Подробнее см. в `docs/index.md`.
